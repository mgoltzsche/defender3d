Die Idee

Die Gruppe hat sich das im Jahre 1980 erschienene Spiel "Defender" ausgesucht.

Der damals auf zwei Dimensionen basierende Weltraumshooter, soll im Laufe dieses Projekts um weitere Features erweitert werden:

  • 3. Dimension
  • Mehrspielermodus
